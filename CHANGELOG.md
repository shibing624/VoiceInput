# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2025-04-07

### Added

- Push-to-talk via Fn key and Right Command key
- Mouse-triggered recording from menu bar
- Live waveform HUD with real-time transcription preview
- CJK input method-aware text injection
- Multi-language support: English, Simplified Chinese, Traditional Chinese, Japanese, Korean
- Optional LLM refinement via OpenAI-compatible API
- On-device speech recognition powered by Apple Speech framework
- DMG packaging via `make dmg`
